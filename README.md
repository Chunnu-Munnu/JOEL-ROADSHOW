<p id="description">Brahmastra is Trinetra’s military module that uses AI to monitor live security feeds instantly detect threats like weapons or intrusions and send real-time alerts for rapid coordinated response. It is built to scale across thousands of cameras for comprehensive defense and minimizes human fatigue and error through automation. This enables proactive large-area military threat prevention.</p>

<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Real-time AI-powered surveillance and instant threat detection across defense areas.
*   Automatic alerts for suspicious activity weapons or intrusions; proactively supports rapid response.
*   Simultaneous monitoring of thousands of CCTV feeds for large-scale coverage.
*   Minimizes human fatigue and error via automation while enabling quick coordinated defense decisions.

<h2>🛠️ Installation Steps:</h2>

<p>1. Install Python 3.11</p>

<p>2. Clone</p>

```
git clone https://github.com/Chunnu-Munnu/VISION_X_TRINETRAA
```

<p>3. Enter the backend folder</p>

```
cd backend
```

<p>4. Create a venv version 3.11</p>

```
py -3.11 venv venv
```

<p>5. Start venv</p>

```
venv\Scripts\activate
```

<p>6. install the dependencies</p>

```
pip install -r requirements.txt
```

<p>7. Run the project locally</p>

```
start test_camera.html
```
